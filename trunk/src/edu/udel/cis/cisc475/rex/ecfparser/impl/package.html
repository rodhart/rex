Responsible for parsing the ECF file and storing the data withing a Config container class.

Details

Any class implementing the ECFParserIF interface will need to be able to open and correctly parse a tex file containing the configuration details for hte exam. It will also need to create an internal representation of parsed file using the interfaces found within the edu.udel.cis.cisc475.rex.config.IF package. @see ConfigIF @see EcfParserIF @see EcfParserFactoryIF
