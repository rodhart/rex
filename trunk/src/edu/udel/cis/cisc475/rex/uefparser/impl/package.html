An instance of UEFParser allows the user to create an ExamIF from a exam tex file. A file provided to UEFParser must use the special latex commands provided by the exam documentclass. This class implements the UEFParserIF and UEFParserFactoryIF interfaces.

Usage

The user of this package can create and parse a file by simply providing the following lines of code:

UEFParserIF parser = new UEFParserFactory();
ExamIF exam = parser.parse("Sample.tex");

For more informations please see the cooresponding classes and methods.
